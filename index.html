Interactive Widget
Peloton's Connected Fitness Subscription Platform. Peloton's massive fixed cost base (content studios, instructors, R&D, SG&A) is largely sunk — it doesn't shrink when subscribers leave. But the cost per subscriber rises as the base erodes, making Peloton less competitive vs. cheaper rivals (Apple Fitness+, NordicTrack), driving further churn. Adjust the sliders and watch the spiral unfold.
📊 Real Data: Default values derived from Peloton 10-K filings (FY2021–FY2025). Fixed costs ≈ subscription COGS + SG&A + R&D. Variable cost ≈ per-subscriber content/support costs. Sources: Peloton SEC filings, MarketBeat, Backlinko.

Cost Recovery Formula
Rate = (Fixed Costs ÷ Subscribers) + Variable Cost
Each period: rate rises → more subscribers churn → fixed costs spread over fewer subs → rate rises again → 🔄 spiral deepens
